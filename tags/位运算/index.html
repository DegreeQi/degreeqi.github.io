【算法】Brian Kernighan算法

Brian Kernighan 算法是指通过将整数 n 与 n-1 进行按位与操作 (n & (n-1))，可以消除 n 二进制表示中最右侧的一个 1。 延申有 n & (~n + 1)提取出整数n最后一位为1的数，如：n: 01101，~n 是将n按位取反就是10010，~n + 1: 10011，最后，n & (~n + 1): 00001 338. Counting Bits 给你一个整数 n ，对于 0 <= i <= n 中的每个 i ，计算其二进制表示中 1 的个数 ，返回一个长度为 n + 1 的数组 ans 作为答案。 示例： 输入：n: 2 输出：[0,1,1] 解释： 0 –> 0 1 –> 1 2 –> 10 最直观的做法是对从 0 到 n 的每个整数直接计算「一比特数」。每个 int 型的数都可以用 32 位二进制数表示，只要遍历其二进制表示的每一位即可得到 1 的数目。 ...
